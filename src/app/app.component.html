Encoding Decoding Result
{{ result.encoding }} {{ result.decoding }} {{ result.result }}

これは何？

  • 文字化けした文字列の復元を色々な組み合わせで試みます。「Encoding」の文字コードでエンコードし、「Decoding」の文字コードでデコードした文字列を「Result」に表示しています。
  • 正常な文字列を与えて、「この文字列が文字化けするとどのように見えるのか」を調べる際にも使えます。
  • 「もじばけらった」インスパイア。iconv-lite と buffer パッケージを利用してフロントエンド・オンリーで動作しています。
